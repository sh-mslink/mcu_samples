@par Example Description 

This project (STEP2 of the System Memory AP example) runs on the AT-START-F415 board.
Before executing this project,user should set system memory to AP mode with ICP tool(STEP1 of the System Memory AP example) .
In this project, codes in main block run usart for printing messages, while codes in System Memory run LED toggle.
